## 5. Структурное представление

### 5.1. Верхнеуровневая компонентная схема

MVP-структура DiaGen включает минимальное ядро CLI и инфраструктуру упаковки:

- `src/diagen/cli.py` - CLI-вход на базе Typer.
- `src/diagen/__init__.py` - метаданные пакета и версия.
- `tests/test_cli.py` - unit-тесты базового поведения CLI.
- `pyproject.toml` - сборка, зависимости, entrypoint `diagen`.
- `.github/workflows/` - CI и релиз в GitHub Releases.

### 5.2. Детализация модулей и сервисов

#### 5.2.1. Модуль CLI (`diagen.cli`)

Ответственность:
- инициализировать приложение Typer;
- предоставить общую справку;
- вывести версию приложения через `--version`.

Публичная точка входа:
- `run()` для console script `diagen`.

##### 5.2.1.1. Расширяемость под плагины

На этапе MVP плагины не реализованы, но выделен отдельный модуль CLI как стабильная оболочка.
В следующих итерациях в ядро будут добавлены:
- контракт плагина;
- загрузчик плагинов;
- диспетчер выполнения скриптов/конвейеров.
