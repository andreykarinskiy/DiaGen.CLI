## 2. Ограничения

### 2.1. Технические ограничения

| №      | Ограничения                                      | Пояснения и предыстория                                                                                                                                                                                                                 |
|--------|--------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `TC01` | Язык реализации - Python                         | Удобно использовать в сценариях автоматизации                                                                                                                                                                                           |
| `TC02` | Способ распространения утилиты - GitHub Releases | Так как GitHub Packages на данный момент не поддерживает PyPi, то придется публиковать Python-пакеты в GitHub Releases, а потом устанавливать их с помощью pip по прямой ссылке на релиз (например, pip install git+https://github.com) |
