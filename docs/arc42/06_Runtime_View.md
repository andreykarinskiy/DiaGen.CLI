## 6. Динамическое представление

### 6.1. Сценарий: запрос версии

1. Пользователь выполняет `diagen --version`.
2. Typer вызывает callback опции `--version`.
3. CLI выводит `DiaGen <version>` в stdout.
4. Приложение завершается с кодом `0`.

### 6.2. Сценарий: запрос справки

1. Пользователь выполняет `diagen --help`.
2. Typer формирует встроенную справку по приложению.
3. CLI возвращает код завершения `0`.

### 6.3. Сценарий: запуск без параметров

1. Пользователь выполняет `diagen`.
2. Срабатывает базовый callback приложения.
3. CLI выводит короткое сообщение о готовности и предлагает использовать `--help`.
